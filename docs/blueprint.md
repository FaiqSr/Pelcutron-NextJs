# **App Name**: AquaPel

## Core Features:

- Navigation Bar: Displays the logo and navigation links: Features, Monitoring Boat, Dashboard, Contact, Login and Get Started.
- Hero Section: Presents the main value proposition with a title, description, call-to-action buttons, and statistics on data points and happy shrimps. The original code for image placeholders is retained.
- Features Section: Highlights the key features of the shrimp farming automation system, retaining image placeholders from original code.
- Monitoring Boat Section: Details the IoT-based monitoring boat with specifications, sensor suite, and integrated design, including image placeholders as standard divs.
- Dashboard Preview Section: Displays a preview of the dashboard monitoring interface, including metrics and recent activity. Retains placeholder styling.
- Call to Action Section: Encourages users to order the E-shrimp System. Retains image placeholders and styling from original code.
- Footer Section: Provides copyright information, product links, company links, and quick contact information.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) for a sense of trust and innovation.
- Background color: Very light gray (#F9FAFA) for a clean and modern look.
- Accent color: Teal (#2DD4BF) to highlight interactive elements and call-to-actions.
- Font: 'Instrument Sans', sans-serif for a modern and readable interface. Note: currently only Google Fonts are supported.
- Use simple, outlined icons for key features and navigation.
- Maintain a clean, spacious layout with consistent padding and margins. Use a maximum width of 1280px for main content.
- Incorporate subtle transitions and animations for a smooth user experience, such as hover effects on links and buttons.
